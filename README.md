# TALESHOP

Демо‑витрина гейминга с каталогом игр, Keycloak‑авторизацией, интеграцией с Telegram‑ботом и встроенной админкой. Проект подготовлен для портфолио и демонстраций клиентам: аккуратный UI, чистая навигация и готовые сценарии.

## Фичи
- Каталог игр с фильтрами, корзиной и оформлением покупки.
- Единый вход через Keycloak с распределением ролей и проверкой доступа к админке.
- Админ‑панель с карточками действий: товары, изображения, бот, пользователи, статистика.
- Интеграция с Telegram‑ботом для уведомлений и быстрой поддержки.
- Чат‑бот на главной странице для ассистирования пользователям.
- Адаптивный интерфейс и единый набор кнопок/отступов без демо-заглушек.

## Стек
- **Backend:** ASP.NET Core
- **Frontend:** React + TypeScript, Tailwind CSS
- **Auth:** Keycloak
- **Infra:** Docker, Docker Compose

## Как запустить
```bash
docker compose up --build
```
После старта приложение доступно на `http://localhost/`. При необходимости импортируйте realm-конфигурацию из папки `keyckoak settings (temp)` перед использованием админки.

## Демо-аккаунты
- admin@taleshop.local — роль admin (используйте тестовый пароль, например `Demo#123`)
- user@taleshop.local — обычный пользователь (тестовый пароль, например `Demo#123`)

## Примечания
- Панель администратора доступна только пользователям с ролью `admin` в приложении `tale-shop-app` (Keycloak).
- Для почтовых или личных данных используйте только демо‑значения, реальные контакты в интерфейсе не отображаются.
